## 專案概述

### 背景與動機

現代人生活忙碌，面對工作、學業、感情、財務等各方面的壓力，時常希望能尋求一些心靈上的慰藉與方向指引。傳統的算命、占卜需要親自到廟宇或找算命師，既耗時又不方便。本系統旨在打造一個 **線上算命平台**，讓使用者隨時隨地都能透過網路進行算命、占卜、抽籤等活動，獲得心靈上的指引與安慰。

### 目標用戶

| 用戶類型 | 描述 |
| -------- | ---- |
| 一般大眾 | 對算命、占卜有興趣，希望隨時能線上求籤問卜的使用者 |
| 年輕族群 | 喜歡趣味性、互動性強的占卜體驗（如每日運勢、塔羅牌等） |
| 信仰族群 | 習慣到廟宇拜拜求籤，希望能線上也能進行類似儀式的使用者 |

### 核心價值主張

- **隨時隨地**：不受時間與地點限制，打開網頁即可算命占卜
- **多元占卜方式**：提供抽籤、塔羅牌、每日運勢等多種算命方式
- **結果保存**：使用者可以保存每次的算命結果，方便日後回顧
- **趣味互動**：透過精美的動畫與互動效果，提升占卜體驗
- **線上香油錢**：讓使用者可以表達心意，支持系統營運

## 功能需求

### 功能一：線上抽籤（求籤問卜）

> 作為 **一般使用者**，我希望 **能在線上進行抽籤（如觀音靈籤）**，以便 **在遇到疑惑時獲得神明的指引與建議**。

**詳細說明：**
- 使用者可選擇籤種類型（如觀音靈籤、月老籤、財神籤等）
- 點擊「抽籤」後顯示搖籤動畫
- 顯示籤詩內容、籤號、吉凶、解籤說明
- 每支籤包含：籤詩、解曰、仙機（工作、感情、財運、健康等面向的建議）

### 功能二：塔羅牌占卜

> 作為 **對塔羅有興趣的使用者**，我希望 **能進行線上塔羅牌占卜**，以便 **了解自己在特定問題上的運勢與建議**。

**詳細說明：**
- 提供多種牌陣選擇（如單張牌、三張牌：過去/現在/未來）
- 使用者心中默念問題後翻牌
- 顯示翻出的塔羅牌圖案與牌義解讀
- 提供正位與逆位的不同解釋

### 功能三：每日運勢

> 作為 **日常使用者**，我希望 **每天都能查看當日的綜合運勢**，以便 **做為一天的參考與心靈指引**。

**詳細說明：**
- 根據使用者的生日或星座顯示當日運勢
- 運勢涵蓋：整體運、工作運、愛情運、財運、健康運
- 以星級（1~5 星）呈現各項運勢指數
- 提供每日幸運色、幸運數字、幸運方位
- 附帶一句「今日箴言」作為心靈指引

### 功能四：算命結果保存與歷史紀錄

> 作為 **註冊使用者**，我希望 **能保存每次的算命結果**，以便 **日後回顧並追蹤自己的運勢變化**。

**詳細說明：**
- 使用者可註冊帳號 / 登入
- 每次算命、占卜、抽籤的結果自動存入個人歷史紀錄
- 提供「我的紀錄」頁面，可依日期、類型篩選
- 可對紀錄加上個人備註（例如：問的什麼問題、事後是否應驗）
- 可刪除不需要的紀錄

### 功能五：捐香油錢（線上打賞）

> 作為 **虔誠的使用者**，我希望 **能線上捐獻香油錢**，以便 **表達對神明的敬意並支持平台營運**。

**詳細說明：**
- 提供固定金額選項（如 $50、$100、$500、$1000）及自訂金額
- 記錄捐獻紀錄（日期、金額、祈福留言）
- 捐獻後顯示感謝頁面與祈福動畫
- 提供「功德榜」頁面顯示捐獻排行（可匿名）

### 功能六：使用者帳號管理

> 作為 **使用者**，我希望 **能註冊並管理我的帳號**，以便 **保存我的個人偏好與歷史紀錄**。

**詳細說明：**
- 註冊功能（帳號、密碼、暱稱、生日）
- 登入 / 登出功能
- 個人資料編輯（修改暱稱、生日、密碼）
- 帳號頁面顯示：個人資訊、算命次數統計、捐獻總額

### 功能七：首頁與導覽

> 作為 **訪客**，我希望 **首頁能清楚展示所有功能入口**，以便 **快速找到想要的算命方式**。

**詳細說明：**
- 首頁展示所有算命功能的入口卡片（抽籤、塔羅、每日運勢）
- 顯示「今日推薦」功能區塊
- 響應式設計，手機與電腦都能使用
- 導覽列（Navbar）包含所有主要頁面的連結

## 非功能需求

### 技術限制

| 項目 | 規格 |
| ---- | ---- |
| 後端框架 | Python + Flask |
| 前端模板 | Jinja2（Flask 內建模板引擎） |
| 資料庫 | SQLite（輕量級，適合練習） |
| 前端樣式 | HTML + CSS + JavaScript |
| 版本控制 | Git + GitHub |

### 效能考量

- 頁面載入時間應在 **3 秒內**
- 資料庫查詢應有適當的索引，避免大量資料時效能下降
- 圖片與動畫資源應進行適當壓縮

### 安全考量

- 使用者密碼需進行 **雜湊加密**（如 bcrypt / werkzeug）儲存
- 防範 **SQL Injection**：使用 ORM 或參數化查詢
- 防範 **XSS**：Jinja2 預設會跳脫 HTML 特殊字元
- 表單提交需加入 **CSRF Token** 保護

### 可用性考量

- 介面文字以 **繁體中文** 為主
- 支援 **響應式設計（RWD）**，適配手機與電腦
- 配色以溫暖、神秘的風格為主（如深紫、金色、暗紅）
- 關鍵操作需有 **動畫反饋**（如抽籤搖動、翻牌動畫）

## MVP 範圍

### 🔴 Must Have（核心功能，必須完成）

| 功能 | 說明 |
| ---- | ---- |
| 線上抽籤 | 至少提供一種籤種（如觀音靈籤），可抽籤並顯示籤詩內容 |
| 每日運勢 | 根據星座或生日顯示當日運勢，含各項指數 |
| 使用者註冊/登入 | 基本的帳號系統，支援註冊、登入、登出 |
| 算命結果保存 | 登入後可自動保存結果，可在「我的紀錄」頁面檢視 |
| 首頁與導覽 | 清楚的首頁入口與導覽列 |

### 🟡 Should Have（重要功能，盡量完成）

| 功能 | 說明 |
| ---- | ---- |
| 塔羅牌占卜 | 提供基本的單張或三張牌占卜功能 |
| 捐香油錢 | 提供捐獻功能與紀錄，含感謝頁面 |
| 抽籤動畫 | 抽籤時顯示搖籤筒動畫，增加趣味性 |

### 🟢 Nice to Have（加分功能，時間允許再做）

| 功能 | 說明 |
| ---- | ---- |
| 功德榜 | 顯示捐獻排行榜，可選擇匿名 |
| 多種籤種 | 提供觀音靈籤以外的其他籤種選擇 |
| 算命紀錄備註 | 使用者可對歷史紀錄加上個人備註 |
| 翻牌動畫 | 塔羅牌占卜時的精美翻牌動畫效果 |
| 社群分享 | 將算命結果分享到社群媒體 |
